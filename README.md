# A script for add the ADOP on your fastfetch 
![exemple](asset/exemple.png)

## Usages 
* add your own keys
  
if you want the script run on startup use the script nasa.sh
```sh
chmod +x ~/.config/pathtothescript/start_nasa.sh
```
if you use sway add this to your sway/config
```sh
exec ~/.config/fastfetch/start_nasa.sh
```
